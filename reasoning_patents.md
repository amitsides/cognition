# Reasoning Patents

This document details a collection of 6 patents focusing on various aspects of reasoning in cognitive systems and AI.

## 1. US10839302B2
**Title:** System for commonsense reasoning using knowledge graphs
**Assignee:** Microsoft
**Year:** 2020
**Domain:** reasoning_engines
**Abstract:** Graph-based commonsense reasoning engine integrating structured knowledge with neural inference.
**Formalizable Claims:**
*   graph_traversal_inference
*   commonsense_entailment
*   knowledge_triple_scoring

## 2. US11250318B2
**Title:** Analogical reasoning in neural-symbolic systems
**Assignee:** IBM
**Year:** 2022
**Domain:** reasoning_engines
**Abstract:** Hybrid system performing analogical reasoning by mapping structural relations between neural embeddings.
**Formalizable Claims:**
*   structure_mapping_function
*   relational_similarity_metric
*   analogical_transfer_operator

## 3. US11157804B2
**Title:** Graph neural networks for abstract reasoning
**Assignee:** DeepMind Technologies
**Year:** 2021
**Domain:** reasoning_engines
**Abstract:** Graph neural network performing abstract visual reasoning over relational structures in perception tasks.
**Formalizable Claims:**
*   message_passing_function
*   relational_inference_step
*   abstract_pattern_completion

## 4. US10540601B2
